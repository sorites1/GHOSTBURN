# Windy

**Cause:** When strong gusts of wind interfere with , it is \[Windy\].

**Effect:** When it is \[Windy\] and you use a Firearms trick to \[[[Keywords/Keyword_Attack|Attack]]\] a target that is \[[[Keywords/Keyword_Far|Far]]\] away (or beyond \[Far\] if using a \[[[Mods_Scope|Scope]]\]), raise the DL by one. Also, when it is \[Windy\] and you use the Athletics trick, [[Skills_and_Tricks/Trick_Athletics_Core_Throw|Throw]], raise the DL by one.

**Removal:** When the wind dies down, it is no longer \[Windy\].
