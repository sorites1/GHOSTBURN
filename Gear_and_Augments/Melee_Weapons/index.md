# Melee Weapons

Melee weapons have a Base Value which is added to the character's Strength score when calculating its \[[[Playing_the_Game/Rolling_the_Dice/Damage_Array|Damage Array]]\], as shown below. 

| Minimum Damage        | Normal Damage               | Maximum Damage              |
| --------------------- | --------------------------- | --------------------------- |
| Strength + Base Value | (Strength + Base Value) x 2 | (Strength + Base Value) x 5 |

For example, a character with a Strength score of 3 is wielding a Switchblade, which has a Base Value of 1. \[Minimum Damage\] is 4, \[Normal Damage\] is 8, and \[Maximum Damage\] is 20. The [[Melee_Strike|Melee Strike]] \[Damage Array\] for this weapon when used by this character is 4/8/20.
