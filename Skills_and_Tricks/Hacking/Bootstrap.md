# Bootstrap \[Signature Trick\]

| Difficulty Level (DL)                               | Skill   | Keywords                                                                                                                                                                                                                                                                                                               |
| --------------------------------------------------- | ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| \[[[Lexicon/Keywords/Keyword_Security\|Security]]\] | Hacking | \[[[Lexicon/Keywords/Keyword_Cyber_Rig\|Cyber Rig]]\], \[[[Lexicon/Keywords/Keyword_Physical\|Physical]]\], \[[[Lexicon/Keywords/Keyword_Range\|Range]]\], \[[[Lexicon/Keywords/Keyword_Two_Hands\|Two Hands]]\], \[[[Lexicon/Keywords/Keyword_Visual\|Visual]]\], \[[[Lexicon/Keywords/Keyword_Wireless\|Wireless]]\] |

*Having already gained Admin access on the device, you analyze its outgoing network traffic and search for a weakness. Once identified, you write a bootstrap script to exploit it, so you can gain control of the system to which the device belongs.*  

To use Bootstrap, you must already have \[[[Lexicon/Keywords/Keyword_Admin|Admin]]\] access on a device, and the device must be connected to a system. The device on which you have \[Admin\] access must be \[[[Lexicon/Keywords/Keyword_Close|Close]]\] or \[[[Lexicon/Keywords/Keyword_Nearby|Nearby]]\]. 

| Outcome      | Description                                                                                                                                                         |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Cool Success | You are a \[Sysadmin\] on the connected system and gain a \[[[Lexicon/Keywords/Keyword_Secret\|Secret]]\]                                                           |
| Success      | You are a \[[[Lexicon/Keywords/Keyword_Sysadmin\|Sysadmin]]\] on the connected system                                                                               |
| Fade         | You are a \[Sysadmin\] on the connected system and have \[[[Lexicon/Conditions/Condition_Stress\|Stress]]\] vs. \[[[Lexicon/Keywords/Keyword_Security\|Security]]\] |
| Glitch       | You lose access to the device and trigger a \[[[Lexicon/Keywords/Keyword_Security_Alert\|Security Alert]]\]                                                         |
