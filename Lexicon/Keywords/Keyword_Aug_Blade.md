# Aug Blade

\[Aug Blade] is a \[[[Lexicon/Keywords/Keyword_Weapon_Types|Weapon Type]]\]. It belongs to the \[[[Lexicon/Keywords/Keyword_Blades|Bladed]]\] weapon group. The \[Aug Blade\] weapon type includes the following keywords: \[[[Lexicon/Keywords/Keyword_Brawl|Brawl]]\], \[[[Lexicon/Keywords/Keyword_Melee|Melee]]\], \[[[Lexicon/Keywords/Keyword_One_Hand|One Hand]]\], and \[[[Lexicon/Keywords/Keyword_Physical|Physical]]\]. 

Weapons with this \[Weapon Type\] use the Brawl skill to \[[[Lexicon/Keywords/Keyword_Attack|Attack]]\], but they use the Melee [[Playing_the_Game/Rolling_the_Dice/Damage_Array|Damage Array]] and can only use tricks from the Melee skill. 

For example, the [[TGR_Claws|TGR Claws]] augment is classified as an \[Aug Blade\]. To make an \[Attack\], you would [[Playing_the_Game/Rolling_the_Dice/Making_a_Roll|Make a Roll]] using Brawl, but you would not use the [[Unarmed_Strike|Unarmed Strike]] trick. Instead, you would choose from the tricks found in the Melee skill, such as [[Melee_Strike|Melee Strike]], [[Dazzling_Strike|Dazzling Strike]], or [[Leaping_Strike|Leaping Strike]].
