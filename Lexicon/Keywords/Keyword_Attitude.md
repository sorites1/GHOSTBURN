# Attitude

\[Attitude\] is an NPC trait. It represents how an NPC feels about a particular person. \[Attitude\] can be thought of as a spectrum with three values: \[[[Lexicon/Keywords/Keyword_Friendly|Friendly]]\], \[[[Lexicon/Keywords/Keyword_Neutral|Neutral]]\], and \[[[Lexicon/Keywords/Keyword_Hostile|Hostile]]\]. 

Some tricks cause a target's \[Attitude\] to improve or worsen. When an \[Attitude\] improves, it goes from \[Hostile\] to \[Neutral\] to \[Friendly\]. When it worsens, it goes from \[Friendly\] to \[Neutral\] to \[Hostile\].
