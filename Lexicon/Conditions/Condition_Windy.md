# Windy

**Cause:** When strong gusts of wind interfere with , it is \[Windy\].

**Effect:** When it is \[Windy\] and you use a Firearms trick to \[[[Lexicon/Keywords/Keyword_Attack|Attack]]\] a target that is \[[[Lexicon/Keywords/Keyword_Far|Far]]\] away (or beyond \[Far\] if using a \[[[Scope|Scope]]\]), raise the DL by one. Also, when it is \[Windy\] and you use the Athletics trick, [[Throw|Throw]], raise the DL by one.

**Removal:** When the wind dies down, it is no longer \[Windy\].
