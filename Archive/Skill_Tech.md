# Tech (Intelligence)

The Tech skill covers drones, robots, repair, research and design

* Run Diagnostics - Run a test to see what is wrong with a gun, drone, robot, or vehicle
* Repair Device - Repair damaged guns, drones, robots, and vehicles
* Install Enhancement - Install an enhancement to a gun, drone, robot, or vehicle
* Disable Device - Temporarily or permanent disable an electronic or mechanical device
* Build Prototype - Build a new enhancement or new model of a gun, drone, robot, or vehicle* 
* Pilot Drone - Use a drone to perform complex maneuvers and use advanced features
