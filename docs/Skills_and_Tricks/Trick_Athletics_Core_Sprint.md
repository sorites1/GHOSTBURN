# Sprint \[Core Trick\]

| Difficulty Level (DL) | Skill     | Keywords                                      |
| --------------------- | --------- | --------------------------------------------- |
| Moderate (14)         | Athletics | [[Move](../Keywords/Keyword_Move\.md)], [[Physical](../Keywords/Keyword_Physical\.md)], [[Range](../Keywords/Keyword_Range\.md)], [[Visual](../Keywords/Keyword_Visual\.md)] |

*You take off in a beeline, putting all your energy into running as fast as you can.* 

When you use Sprint, you must [[Move](../Keywords/Keyword_Move.md)] in a straight line.

| Outcome      | Description                                                                                                                                   |
| ------------ | --------------------------------------------------------------------------------------------------------------------------------------------- |
| Cool Success | You \[Move\] \[Far\] and recover 1 AP                                                                                                         |
| Success      | You \[Move\] \[Far\]                                                                                                                          |
| Fade         | You \[Move\] [[Far](../Keywords/Keyword_Far\.md)] and are \[Off Balance\]                                                                        |
| Glitch       | You [[Move](../Keywords/Keyword_Move\.md)] [[Nearby](../Keywords/Keyword_Nearby\.md)] and are [[Off Balance](../Conditions/Condition_Off_Balance\.md)] |
