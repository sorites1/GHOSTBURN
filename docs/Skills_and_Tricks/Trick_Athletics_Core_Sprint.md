# Sprint \[Core Trick\]

| Difficulty Level (DL) | Skill     | Keywords                                      |
| --------------------- | --------- | --------------------------------------------- |
| Moderate (14)         | Athletics | [[Move](/../Keywords/Keyword_Move)], [[Physical](/../Keywords/Keyword_Physical)], [[Range](/../Keywords/Keyword_Range)], [[Visual](/../Keywords/Keyword_Visual)] |

*You take off in a beeline, putting all your energy into running as fast as you can.* 

When you use Sprint, you must [[Move](/../Keywords/Keyword_Move)] in a straight line.

| Outcome      | Description                                                                                                                                   |
| ------------ | --------------------------------------------------------------------------------------------------------------------------------------------- |
| Cool Success | You \[Move\] \[Far\] and recover 1 AP                                                                                                         |
| Success      | You \[Move\] \[Far\]                                                                                                                          |
| Fade         | You \[Move\] [[Far](/../Keywords/Keyword_Far)] and are \[Off Balance\]                                                                        |
| Glitch       | You [[Move](/../Keywords/Keyword_Move)] [[Nearby](/../Keywords/Keyword_Nearby)] and are [[Off Balance](/../Conditions/Condition_Off_Balance)] |
