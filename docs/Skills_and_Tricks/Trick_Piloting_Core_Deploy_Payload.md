# Deploy Payload \[Core Trick\]

| Difficulty Level (DL) | Skill    | Keywords                                                                                                                              |
| --------------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| Moderate (14)         | Piloting | [[Concentrate](../Keywords/Keyword_Concentrate\.md)], [[Control Deck](../Keywords/Keyword_Control_Deck\.md)], [[Datajack](../Keywords/Keyword_Datajack\.md)], [[Drone](../Keywords/Keyword_Drone\.md)], [[Fly](../Keywords/Keyword_Fly\.md)], [[Range](../Keywords/Keyword_Range\.md)], [[Physical](../Keywords/Keyword_Physical\.md)], [[Two Hands](../Keywords/Keyword_Two_Hands\.md)], [[Visual](../Keywords/Keyword_Visual\.md)], [[Wireless](../Keywords/Keyword_Wireless\.md)] |

*Having piloted your drone to its destination, you carefully position it and deploy its payload.*

To use Deploy Payload, describe the spot [[Nearby](../Keywords/Keyword_Nearby.md)] where you want to drop your payload. This spot is called *the mark*. For example, *I want to place the package in front of the door*, or *I want to drop the grenade on the truck*. If the mark is in a spot that is hard to navigate, the the GM should raise the DL by one. If the mark is extremely hard to navigate, raise the DL by two.

| Outcome      | Description                                                                                                                                 |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------- |
| Cool Success | You \[Fly\] the \[Drone\] \[Nearby\], hit the mark, and are [[Inspired](../Conditions/Condition_Inspired\.md)]                                 |
| Success      | You \[Fly\] the \[Drone\] \[Nearby\] and hit the mark                                                                                       |
| Fade         | You \[Fly\] the \[Drone\] \[Nearby\] but fail to deploy your payload and gain [[Stress](../Conditions/Condition_Stress\.md)] vs. Moderate (14) |
| Glitch       | You \[Fly\] the \[Drone\] [[Close](../Keywords/Keyword_Close\.md)], miss the mark, and have a [[Bad Memory](../Keywords/Keyword_Bad_Memory\.md)]  |
