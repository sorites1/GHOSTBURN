# Deploy Payload \[Core Trick\]

| Difficulty Level (DL) | Skill    | Keywords                                                                                                                              |
| --------------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| Moderate (14)         | Piloting | [[Concentrate](/../Keywords/Keyword_Concentrate)], [[Control Deck](/../Keywords/Keyword_Control_Deck)], [[Datajack](/../Keywords/Keyword_Datajack)], [[Drone](/../Keywords/Keyword_Drone)], [[Fly](/../Keywords/Keyword_Fly)], [[Range](/../Keywords/Keyword_Range)], [[Physical](/../Keywords/Keyword_Physical)], [[Two Hands](/../Keywords/Keyword_Two_Hands)], [[Visual](/../Keywords/Keyword_Visual)], [[Wireless](/../Keywords/Keyword_Wireless)] |

*Having piloted your drone to its destination, you carefully position it and deploy its payload.*

To use Deploy Payload, describe the spot [[Nearby](/../Keywords/Keyword_Nearby)] where you want to drop your payload. This spot is called *the mark*. For example, *I want to place the package in front of the door*, or *I want to drop the grenade on the truck*. If the mark is in a spot that is hard to navigate, the the GM should raise the DL by one. If the mark is extremely hard to navigate, raise the DL by two.

| Outcome      | Description                                                                                                                                 |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------- |
| Cool Success | You \[Fly\] the \[Drone\] \[Nearby\], hit the mark, and are [[Inspired](/../Conditions/Condition_Inspired)]                                 |
| Success      | You \[Fly\] the \[Drone\] \[Nearby\] and hit the mark                                                                                       |
| Fade         | You \[Fly\] the \[Drone\] \[Nearby\] but fail to deploy your payload and gain [[Stress](/../Conditions/Condition_Stress)] vs. Moderate (14) |
| Glitch       | You \[Fly\] the \[Drone\] [[Close](/../Keywords/Keyword_Close)], miss the mark, and have a [[Bad Memory](/../Keywords/Keyword_Bad_Memory)]  |
