# Manipulate Object \[Signature Trick\]

| Difficulty Level (DL) | Skill    | Keywords                                                                                                                               |
| --------------------- | -------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| Hard (20)             | Piloting | [[Manipulator Arm](/../Keywords/Keyword_Manipulator_Arm)], [[Concentrate](/../Keywords/Keyword_Concentrate)], [[Control Deck](/../Keywords/Keyword_Control_Deck)], [[Datajack](/../Keywords/Keyword_Datajack)], [[Drone](/../Keywords/Keyword_Drone)], [[Physical](/../Keywords/Keyword_Physical)], [[Two Hands](/../Keywords/Keyword_Two_Hands)], [[Visual](/../Keywords/Keyword_Visual)], [[Wireless](/../Keywords/Keyword_Wireless)] |

*You use your drone's manipulator arm to open doors, use a keypad, or pick up an object, just like you would use your own.*

To use Manipulate Object, your [[Drone](/../Keywords/Keyword_Drone)] must be outfitted with a [[Manipulator Arm](/../Keywords/Keyword_Manipulator_Arm)]. When you use Manipulate Object, state the action you want to perform, such as *I want use my drone to type the security code into the keypad*, or *I want my drone to pick up the pistol and put it in its cargo bay*. 

| Outcome      | Description                                                                                                 |
| ------------ | ----------------------------------------------------------------------------------------------------------- |
| Cool Success | You use the \[Manipulator Arm\] successfully and you are [[Inspired](/../Conditions/Condition_Inspired)]    |
| Success      | You use the \[Manipulator Arm\] successfully                                                                |
| Fade         | You use the [[Manipulator Arm](/../Keywords/Keyword_Manipulator_Arm)] successfully and you are \[Confused\] |
| Glitch       | You are unsuccessful and [[Confused](/../Conditions/Condition_Confused)]                                    |
