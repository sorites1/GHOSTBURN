# Disable Device \[Core Trick\]

| Difficulty Level (DL)                       | Skill   | Keywords                                                                                                                                                                                                                        |
| ------------------------------------------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [[Security](../Keywords/Keyword_Security\.md)] | Hacking | [[Fine Control](../Keywords/Keyword_Fine_Control\.md)], [[Physical](../Keywords/Keyword_Physical\.md)], [[Tools](../Keywords/Keyword_Tools\.md)], [[Two Hands](../Keywords/Keyword_Two_Hands\.md)], [[Visual](../Keywords/Keyword_Visual\.md)] |

*You inspect the device's electronic guts and make a calculated choice to snip one of the wires. A second later, the maglock disengages, and the door opens with a hiss.* 

Disable Device is a hands-on activity that requires you to directly interact with the target's hardware. If you do not have proper [[Tools](../Keywords/Keyword_Tools.md)], raise the DL by one.

| Outcome      | Description                                                                                                                                             |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Cool Success | The device is \[Disabled\] and you gain a [[Secret](../Keywords/Keyword_Secret\.md)]                                                                       |
| Success      | The device is \[Disabled\]                                                                                                                              |
| Fade         | The device is [[Disabled](../Keywords/Keyword_Disabled\.md)] and you have \[Vision Impairment\]                                                            |
| Glitch       | You fail, have [[Vision Impairment](../Conditions/Condition_Vision_Impairment\.md)], and trigger a [[Security Alert](../Keywords/Keyword_Security_Alert\.md)] |
