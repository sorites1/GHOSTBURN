# Bootstrap \[Signature Trick\]

| Difficulty Level (DL)                               | Skill   | Keywords                                                                                                                                                                                                                                                                                                               |
| --------------------------------------------------- | ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [[Security](../../Lexicon/Keywords/Keyword_Security.md)] | Hacking | [[Cyber Rig](../../Lexicon/Keywords/Keyword_Cyber_Rig.md)], [[Physical](../../Lexicon/Keywords/Keyword_Physical.md)], [[Range](../../Lexicon/Keywords/Keyword_Range.md)], [[Two Hands](../../Lexicon/Keywords/Keyword_Two_Hands.md)], [[Visual](../../Lexicon/Keywords/Keyword_Visual.md)], [[Wireless](../../Lexicon/Keywords/Keyword_Wireless.md)] |

*Having already gained Admin access on the device, you analyze its outgoing network traffic and search for a weakness. Once identified, you write a bootstrap script to exploit it, so you can gain control of the system to which the device belongs.*  

To use Bootstrap, you must already have [[Admin](../../Lexicon/Keywords/Keyword_Admin.md)] access on a device, and the device must be connected to a system. The device on which you have \[Admin\] access must be [[Close](../../Lexicon/Keywords/Keyword_Close.md)] or [[Nearby](../../Lexicon/Keywords/Keyword_Nearby.md)]. 

| Outcome      | Description                                                                                                                                                         |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Cool Success | You are a \[Sysadmin\] on the connected system and gain a [[Secret](../../Lexicon/Keywords/Keyword_Secret.md)]                                                           |
| Success      | You are a [[Sysadmin](../../Lexicon/Keywords/Keyword_Sysadmin.md)] on the connected system                                                                               |
| Fade         | You are a \[Sysadmin\] on the connected system and have [[Stress](../../Lexicon/Conditions/Condition_Stress.md)] vs. [[Security](../../Lexicon/Keywords/Keyword_Security.md)] |
| Glitch       | You lose access to the device and trigger a [[Security Alert](../../Lexicon/Keywords/Keyword_Security_Alert.md)]                                                         |
