# Droptrip \[Signature Trick\]

| Difficulty Level (DL)                       | Skill   | Keywords                                                                                                                                                                                                                              |
| ------------------------------------------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [[Security](../../Lexicon/Keywords/Keyword_Security.md)] | Hacking | [[Cyber Rig](../../Lexicon/Keywords/Keyword_Cyber_Rig.md)], [[Datajack](../../Lexicon/Keywords/Keyword_Datajack.md)], [[Physical](../../Lexicon/Keywords/Keyword_Physical.md)], [[Two Hands](../../Lexicon/Keywords/Keyword_Two_Hands.md)], [[Unaware](../../Lexicon/Conditions/Condition_Unaware.md)] |
