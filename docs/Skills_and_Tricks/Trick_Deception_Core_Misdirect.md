# Misdirect \[Core Trick\]

| Difficulty Level (DL) | Skill     | Keywords                                                                                                                                                                 |
| --------------------- | --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Wits                  | Deception | [[Attitude](../Keywords/Keyword_Attitude\.md)], [[Language](../Keywords/Keyword_Language\.md)], [[One Hand](../Keywords/Keyword_One_Hand\.md)]. [[Social](../Keywords/Keyword_Social\.md)], [[Vocal](../Keywords/Keyword_Vocal\.md)] |

*Whether by shouting and pointing or simply by talking and using one hand, you momentarily distract your opponent.*

To use Misdirect on an opponent, they must be [[Close](../Keywords/Keyword_Close.md)] or [[Nearby](../Keywords/Keyword_Nearby.md)]. If the target has an \[Attitude\] of Hostile towards you, raise the DL by one.

| Outcome      | Description                                                                                 |
| ------------ | ------------------------------------------------------------------------------------------- |
| Cool Success | They are \[Distracted\] and you are [[Inspired](../Conditions/Condition_Inspired\.md)]         |
| Success      | They are \[Distracted\]                                                                     |
| Fade         | They are [[Distracted](../Conditions/Condition_Distracted\.md)] and they become \[Suspicious\] |
| Glitch       | They become \[Suspicious\], and they gain [[Confident](../Conditions/Condition_Confident\.md)] |
