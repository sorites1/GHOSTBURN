# Droptrip \[Signature Trick\]

| Difficulty Level (DL)                       | Skill   | Keywords                                                                                                                                                                                                                              |
| ------------------------------------------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [[Security](../Keywords/Keyword_Security\.md)] | Hacking | [[Cyber Rig](../Keywords/Keyword_Cyber_Rig\.md)], [[Datajack](../Keywords/Keyword_Datajack\.md)], [[Physical](../Keywords/Keyword_Physical\.md)], [[Two Hands](../Keywords/Keyword_Two_Hands\.md)], [[Unaware](../Conditions/Condition_Unaware\.md)] |

*When you enter an address and jack in, your cyber rig overrides your senses and drops you into the network foam, the lowest layer of the Net. By navigating a digital hallucination, you can gain unauthorized access to the remote system.* 

| Outcome      | Description                                                                                                                                                 |
| ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Cool Success | You are an [[Admin](../Keywords/Keyword_Admin\.md)]                                                                                                            |
| Success      | You are an \[Operator\]                                                                                                                                     |
| Fade         | You are an [[Operator](../Keywords/Keyword_Operator\.md)] and have [[Stress](../Conditions/Condition_Stress\.md)] vs. [[Security](../Keywords/Keyword_Security\.md)] |
| Glitch       | You have no access and trigger a [[Security Alert](../Keywords/Keyword_Security_Alert\.md)]                                                                    |
