# Droptrip \[Signature Trick\]

| Difficulty Level (DL) | Skill | Keywords                                                              |
| --------------------- | ----- | --------------------------------------------------------------------- |
| [[Security](/../Keywords/Keyword_Security)]          | Tech  | [[Cyber Rig](/../Keywords/Keyword_Cyber_Rig)], [[Datajack](/../Keywords/Keyword_Datajack)], [[Physical](/../Keywords/Keyword_Physical)], [[Two Hands](/../Keywords/Keyword_Two_Hands)], [[Unaware](/../Conditions/Condition_Unaware)] |

| ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |

| Success      | You are an \[Operator\]                                                                                                                                     |
