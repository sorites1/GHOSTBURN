# Grapple \[Core Trick\]

Without this trick, 

| Difficulty Level (DL)            | Skill | Keywords                                                                                                                                                     |
| -------------------------------- | ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| [[Opposed](/../Keywords/Keyword_Opposed)] | Brawl | [[Attack](/../Keywords/Keyword_Attack)], [[Move](/../Keywords/Keyword_Move)], [[Physical](/../Keywords/Keyword_Physical)], [[Range](/../Keywords/Keyword_Range)], [[Visual](/../Keywords/Keyword_Visual)] |

*You run after your opponent, grabbing them and tacking them to the ground.* 

When you use Grapple, [[Move](/../Keywords/Keyword_Move)] to an opponent who is [[Close](/../Keywords/Keyword_Close)] or [[Nearby](/../Keywords/Keyword_Nearby)] and then roll Grapple.

| Outcome      | Description                                                                                                            |
| ------------ | ---------------------------------------------------------------------------------------------------------------------- |
| Cool Success | They are \[Restrained\], \[Prone\] and [[Shaken](/../Conditions/Condition_Shaken)]                                     |
| Success      | They are \[Restrained\] and \[Prone\]                                                                                  |
| Fade         | They are \[Restrained\] and you are both \[Prone\]                                                                     |
| Glitch       | You are [[Restrained](/../Conditions/Condition_Restrained)] and you are both [[Prone](/../Conditions/Condition_Prone)] |
