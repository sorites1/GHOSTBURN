# Grapple \[Core Trick\]

Without this trick, 

| Difficulty Level (DL)            | Skill | Keywords                                                                                                                                                     |
| -------------------------------- | ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| [[Opposed](../Keywords/Keyword_Opposed\.md)] | Brawl | [[Attack](../Keywords/Keyword_Attack\.md)], [[Move](../Keywords/Keyword_Move\.md)], [[Physical](../Keywords/Keyword_Physical\.md)], [[Range](../Keywords/Keyword_Range\.md)], [[Visual](../Keywords/Keyword_Visual\.md)] |

*You run after your opponent, grabbing them and tacking them to the ground.* 

When you use Grapple, [[Move](../Keywords/Keyword_Move.md)] to an opponent who is [[Close](../Keywords/Keyword_Close.md)] or [[Nearby](../Keywords/Keyword_Nearby.md)] and then roll Grapple.

| Outcome      | Description                                                                                                            |
| ------------ | ---------------------------------------------------------------------------------------------------------------------- |
| Cool Success | They are \[Restrained\], \[Prone\] and [[Shaken](../Conditions/Condition_Shaken\.md)]                                     |
| Success      | They are \[Restrained\] and \[Prone\]                                                                                  |
| Fade         | They are \[Restrained\] and you are both \[Prone\]                                                                     |
| Glitch       | You are [[Restrained](../Conditions/Condition_Restrained\.md)] and you are both [[Prone](../Conditions/Condition_Prone\.md)] |
