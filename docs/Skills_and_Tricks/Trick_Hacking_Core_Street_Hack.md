# Street Hack \[Core Trick\]

| Difficulty Level (DL)                       | Skill   | Keywords                                                                                                                                                                                                                                                               |
| ------------------------------------------- | ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [[Security](/../Keywords/Keyword_Security)] | Hacking | [[Cyber Rig](/../Keywords/Keyword_Cyber_Rig)], [[Physical](/../Keywords/Keyword_Physical)], [[Range](/../Keywords/Keyword_Range)], [[Two Hands](/../Keywords/Keyword_Two_Hands)], [[Visual](/../Keywords/Keyword_Visual)], [[Wireless](/../Keywords/Keyword_Wireless)] |

*Your type in the command and your cyber rig spins up, creating an AR overlay of the world around you. By selecting one of the color-coded devices in your field of view, you execute the hack.* 

To use Street Hack, the device must be [[Close](/../Keywords/Keyword_Close)] or [[Nearby](/../Keywords/Keyword_Nearby)].

| Outcome      | Description                                                                                                                                                 |
| ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Cool Success | You are an [[Admin](/../Keywords/Keyword_Admin)]                                                                                                            |
| Success      | You are an \[Operator\]                                                                                                                                     |
| Fade         | You are an [[Operator](/../Keywords/Keyword_Operator)] and have [[Stress](/../Conditions/Condition_Stress)] vs. [[Security](/../Keywords/Keyword_Security)] |
| Glitch       | You have no access and trigger a [[Security Alert](/../Keywords/Keyword_Security_Alert)]                                                                    |
