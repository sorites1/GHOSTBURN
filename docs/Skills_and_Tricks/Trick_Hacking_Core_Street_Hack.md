# Street Hack \[Core Trick\]

| Difficulty Level (DL)                       | Skill   | Keywords                                                                                                                                                                                                                                                               |
| ------------------------------------------- | ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [[Security](../Keywords/Keyword_Security\.md)] | Hacking | [[Cyber Rig](../Keywords/Keyword_Cyber_Rig\.md)], [[Physical](../Keywords/Keyword_Physical\.md)], [[Range](../Keywords/Keyword_Range\.md)], [[Two Hands](../Keywords/Keyword_Two_Hands\.md)], [[Visual](../Keywords/Keyword_Visual\.md)], [[Wireless](../Keywords/Keyword_Wireless\.md)] |

*Your type in the command and your cyber rig spins up, creating an AR overlay of the world around you. By selecting one of the color-coded devices in your field of view, you execute the hack.* 

To use Street Hack, the device must be [[Close](../Keywords/Keyword_Close.md)] or [[Nearby](../Keywords/Keyword_Nearby.md)].

| Outcome      | Description                                                                                                                                                 |
| ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Cool Success | You are an [[Admin](../Keywords/Keyword_Admin\.md)]                                                                                                            |
| Success      | You are an \[Operator\]                                                                                                                                     |
| Fade         | You are an [[Operator](../Keywords/Keyword_Operator\.md)] and have [[Stress](../Conditions/Condition_Stress\.md)] vs. [[Security](../Keywords/Keyword_Security\.md)] |
| Glitch       | You have no access and trigger a [[Security Alert](../Keywords/Keyword_Security_Alert\.md)]                                                                    |
