# Manipulate Object \[Signature Trick\]

| Difficulty Level (DL) | Skill    | Keywords                                                                                                                               |
| --------------------- | -------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| Hard (20)             | Piloting | [[Manipulator Arm](../../Lexicon/Keywords/Keyword_Manipulator_Arm.md)], [[Concentrate](../../Lexicon/Keywords/Keyword_Concentrate.md)], [[Control Deck](../../Lexicon/Keywords/Keyword_Control_Deck.md)], [[Datajack](../../Lexicon/Keywords/Keyword_Datajack.md)], [[Drone](../../Lexicon/Keywords/Keyword_Drone.md)], [[Physical](../../Lexicon/Keywords/Keyword_Physical.md)], [[Two Hands](../../Lexicon/Keywords/Keyword_Two_Hands.md)], [[Visual](../../Lexicon/Keywords/Keyword_Visual.md)], [[Wireless](../../Lexicon/Keywords/Keyword_Wireless.md)] |

*You use your drone's manipulator arm to open doors, use a keypad, or pick up an object, just like you would use your own.*

To use Manipulate Object, your [[Drone](../../Lexicon/Keywords/Keyword_Drone.md)] must be outfitted with a [[Manipulator Arm](../../Lexicon/Keywords/Keyword_Manipulator_Arm.md)]. When you use Manipulate Object, state the action you want to perform, such as *I want use my drone to type the security code into the keypad*, or *I want my drone to pick up the pistol and put it in its cargo bay*. 

| Outcome      | Description                                                                                                 |
| ------------ | ----------------------------------------------------------------------------------------------------------- |
| Cool Success | You use the \[Manipulator Arm\] successfully and you are [[Inspired](../../Lexicon/Conditions/Condition_Inspired.md)]    |
| Success      | You use the \[Manipulator Arm\] successfully                                                                |
| Fade         | You use the [[Manipulator Arm](../../Lexicon/Keywords/Keyword_Manipulator_Arm.md)] successfully and you are \[Confused\] |
| Glitch       | You are unsuccessful and [[Confused](../../Lexicon/Conditions/Condition_Confused.md)]                                    |
