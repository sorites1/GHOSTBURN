# Windy

**Cause:** When strong gusts of wind interfere with , it is \[Windy\].

**Effect:** When it is \[Windy\] and you use a Firearms trick to [[Attack](../Keywords/Keyword_Attack.md)] a target that is [[Far](../Keywords/Keyword_Far.md)] away (or beyond \[Far\] if using a [[Scope](../Mod_Scope.md)]), raise the DL by one. Also, when it is \[Windy\] and you use the Athletics trick, [Throw](../Skills_and_Tricks/Trick_Athletics_Core_Throw.md), raise the DL by one.

**Removal:** When the wind dies down, it is no longer \[Windy\].
