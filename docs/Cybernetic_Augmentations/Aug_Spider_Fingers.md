# Spider Fingers

The \[Spider Fingers\] augment requires the replacement of all five fingers on each hand. When you make a Hacking roll using your [[Cyber Rig](/../Keywords/Keyword_Cyber_Rig)], your fingers split open, revealing an array of long, thin, metal digits that allow you to type with lightning speed. 

| Points | Augmentation   | Mesh Cost | Reduces Psi | Description                                                 |
| :----: | -------------- | :-------: | ----------- | ----------------------------------------------------------- |
|   1    | Spider Fingers |     1     | Yes         | +1 [[Aug Bonus](/../Keywords/Keyword_Aug_Bonus)] to Hacking |
