# Cybernetic Lips \[Augment\]

*Cybernetic Lips replace your natural lips.*

|   Cost   | Augment                   | Mesh Cost | Mod Slots | Keywords |
| :------: | ------------------------- | :-------: | :-------: | -------- |
| 10,000 ⌆ | Cybernetic Lips (Base)    |     1     |     0     | -        |
| 30,000 ⌆ | Cybernetic Lips (Level 1) |     1     |     1     | -        |
| 50,000 ⌆ | Cybernetic Lips (Level 2) |     1     |     2     | -        |

Cybernetic Lips (Base) are a replacement only. They performs all the functions of natural lips. 

## Augment Mods

The following augments have the [[Mod](../Keywords/Keyword_Mod.md)] keyword and can be installed in a Cybernetic Lips with a free Mod Slot. Each \[Mod\] augment takes one Mod Slot. Mods: [Poison Reservoir](../Aug_Poison_Reservoir.md)

## Presence Mod

You can install a Presence Mod in a free Mod Slot in Cybernetic Lips to permanently raise your Presence, as shown below. You may only install this \[Mod\] once.

|   Cost   | Mod               | Keywords                                                                               |
| :------: | ----------------- | -------------------------------------------------------------------------------------- |
| 60,000 ⌆ | +1 Presence (Aug) | [[Aug Increase](../Keywords/Keyword_Aug_Increase\.md)], [[Mod](../Keywords/Keyword_Mod\.md)] |

You should update your Deception, Intimidate, and Persuade skills to use your new Presence score.
