# Cybernetic Lung \[Augment\]

The \[Cybernetic Lung\] augment replaces your existing, natural lung. \[Cybernetic Lung\] comes in two levels: \[Cybernetic Lung 0\] and \[Cybernetic Lung 1\]. These levels represent an upgrade path; they do not stack.

| Points | Augmentation      | Mesh Cost | Reduces Psi | Description         |
| :----: | ----------------- | :-------: | ----------- | ------------------- |
|   4    | Cybernetic Lung 0 |     4     | No          | Functions as a lung |
|   6    | Cybernetic Lung 1 |     4     | No          | Can accept one Mod  |
|   8    | Cybernetic Lung 2 |     4     | No          | Can accept two Mods |

The following \[Mods\] are available for Cybernetic Lung.

| Mod                | Description                                                                                     |
| ------------------ | ----------------------------------------------------------------------------------------------- |
| Oxygen Reservoir   | If you would gain [[Drowning](/../Conditions/Condition_Drowning)], you do not gain \[Drowning\] |
| Assistive Breather | You gain a +1 [[Aug Bonus](/../Keywords/Keyword_Aug_Bonus)] to Athletics                        |
