# Long Blade

\[Long Blade] is a [[Weapon Type](../Keywords/Keyword_Weapon_Types.md)]. It belongs to the [[Bladed](../Keyword_Blades.md)] weapon group. The \[Long Blade\] weapon type includes the following keywords: [[Melee](../Keywords/Keyword_Melee.md)] and [[Two Hands](../Keywords/Keyword_Two_Hands.md)].
