# Weapon Groups

A \[Weapon Group\] is a collection of [[Weapon Types](/../Keywords/Keyword_Weapon_Types)]. 

There are four \[Weapon Groups\]: [[Bladed](/../Keywords/Keyword_Bladed)], [[Blunt Force](/../Keywords/Keyword_Blunt_Force)], [[Light Arms](/../Keywords/Keyword_Light_Arms)], and [[Heavy Arms](/../Keywords/Keyword_Heavy_Arms)]. 

Weapon Groups are used by [Armor](/../Keywords/Keyword_Armor).
