# Weapon Groups

A \[Weapon Group\] is a collection of [[Weapon Types](../Keywords/Keyword_Weapon_Types.md)]. 

There are four \[Weapon Groups\]: [[Blades](../Keyword_Blades.md)], [[Blunt Force](../Keywords/Keyword_Blunt_Force.md)], [[Small Arms](../Keyword_Small_Arms.md)], and [[Rifles](../Keywords/Keyword_Rifles.md)]. 

Weapon Groups are used by [Armor](../Keywords/Keyword_Armor.md).
