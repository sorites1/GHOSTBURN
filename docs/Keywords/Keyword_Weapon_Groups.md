# Weapon Groups

A \[Weapon Group\] is a collection of [[Weapon Types](/../Keywords/Keyword_Weapon_Types)]. 

There are four \[Weapon Groups\]: [[Blades](/../Keyword_Blades)], [[Blunt Force](/../Keywords/Keyword_Blunt_Force)], [[Small Arms](/../Keyword_Small_Arms)], and [[Rifles](/../Keywords/Keyword_Rifles)]. 

Weapon Groups are used by [Armor](/../Keywords/Keyword_Armor).
