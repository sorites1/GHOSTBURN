# Unarmed Strike

Indicates an [[Attack](/../Keywords/Keyword_Attack)] made without weapons, such as a punch or kick. See the [[Unarmed Strike](/../Skills_and_Tricks/Trick_Brawl_Core_Unarmed_Strike)] trick for more information. 

The [[Damage Array](/../Keywords/Keyword_Damage_Array)] for \[Unarmed Strike\] uses your Strength score for [[Minimum Damage](/../Keywords/Keyword_Minimum_Damage)], Strength x 2 for [[Normal Damage](/../Keywords/Keyword_Normal_Damage)], and Strength x 5 for [[Maximum Damage](/../Keywords/Keyword_Maximum_Damage)].
