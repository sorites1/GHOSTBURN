# Weapon Types

Each weapon has a \[Weapon Type\] which grants it certain characteristics, as shown in the chart below. There are eight \[Weapon Types\] are Blade, Club, Heavy Pistol, Light Pistol, Rifle, Shotgun, SMG, and Unarmed. Each \[Weapon Type\] belongs to a [[Weapon Group](/../Keyword_Weapon_Groups)].

| Weapon Type                                         | Weapon Group | Weapon Type Characteristics                                                                                                                                                                                                                 |
| --------------------------------------------------- | ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Blade                                               | Bladed       |                                                                                                                                                                                                                                             |
| Club                                                | Blunt Force  |                                                                                                                                                                                                                                             |
| [[Heavy Pistol](/../Keywords/Keyword_Heavy_Pistol)] | Heavy Arms   | [[Firearm](/../Keywords/Keyword_Firearm)], [[Heavy Arms](/../Keywords/Keyword_Heavy_Arms)], [[One Hand](/../Keywords/Keyword_One_Hand)], [[Recoil](/../Keywords/Keyword_Recoil)], [[Two-Handed Grip](/../Keywords/Keyword_Two_Handed_Grip)] |
| [[Light Pistol](/../Keywords/Keyword_Light_Pistol)] | Light Arms   | [[Concealable](/../Keywords/Keyword_Concealable)], \[Firearm\], \[One Hand\], \[Two-Handed Grip\]                                                                                                                                           |
| [[Rifle](/../Keywords/Keyword_Rifle)]               | Heavy Arms   | \[Firearm\], \[Recoil\], [[Two Hands](/../Keywords/Keyword_Two_Hands)]                                                                                                                                                                      |
| [[Shotgun](/../Keywords/Keyword_Shotgun)]           | Light Arms   | \[Firearm\], \[Recoil\], [[Spread](/../Keywords/Keyword_Spread)], \[Two Hands\]                                                                                                                                                             |
| [[SMG](/../Keywords/Keyword_SMG)]                   | Light Arms   | \[Firearm\], \[One Hand\], \[Recoil\], [[Spray](/../Keywords/Keyword_Spray)]                                                                                                                                                                |
| Unarmed                                             | Blunt Force  |                                                                                                                                                                                                                                             |
