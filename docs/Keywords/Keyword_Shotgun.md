# Shotgun

\[Shotgun\] is a [[Weapon Type](../Keywords/Keyword_Weapon_Types.md)]. It belongs to the [[Light Arms](../Keyword_Small_Arms.md)] weapon group. The \[Shotgun\] weapon type includes the following keywords: [[Firearm](../Keywords/Keyword_Firearm.md)], [[Recoil](../Keywords/Keyword_Recoil.md)], [[Spread](../Keywords/Keyword_Spread.md)], and [[Two Hands](../Keywords/Keyword_Two_Hands.md)].
