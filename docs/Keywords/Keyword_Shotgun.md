# Shotgun

\[Shotgun\] is a [[Weapon Type](/../Keywords/Keyword_Weapon_Types)]. It belongs to the [[Light Arms](/../Keywords/Keyword_Light_Arms)] weapon group. The \[Shotgun\] weapon type includes the following keywords: [[Firearm](/../Keywords/Keyword_Firearm)], [[Recoil](/../Keywords/Keyword_Recoil)], [[Spread](/../Keywords/Keyword_Spread)], and [[Two Hands](/../Keywords/Keyword_Two_Hands)].
