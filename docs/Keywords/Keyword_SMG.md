# SMG

\[SMG\] is a [[Weapon Type](../Keywords/Keyword_Weapon_Types.md)]. It belongs to the [[Light Arms](../Keyword_Small_Arms.md)] weapon group. The \[SMG\] weapon type includes the following keywords: [[Firearm](../Keywords/Keyword_Firearm.md)], [[One Hand](../Keywords/Keyword_One_Hand.md)], [[Recoil](../Keywords/Keyword_Recoil.md)], and [[Spray](../Keywords/Keyword_Spray.md)].
