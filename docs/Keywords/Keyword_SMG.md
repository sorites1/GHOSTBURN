# SMG

\[SMG\] is a [[Weapon Type](/../Keywords/Keyword_Weapon_Types)]. It belongs to the [[Light Arms](/../Keywords/Keyword_Light_Arms)] weapon group. The \[SMG\] weapon type includes the following keywords: [[Firearm](/../Keywords/Keyword_Firearm)], [[One Hand](/../Keywords/Keyword_One_Hand)], [[Recoil](/../Keywords/Keyword_Recoil)], and [[Spray](/../Keywords/Keyword_Spray)].
