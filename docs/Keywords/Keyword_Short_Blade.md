# Short Blade

\[Short Blade] is a [[Weapon Type](/../Keywords/Keyword_Weapon_Types)]. It belongs to the [[Bladed](/../Keyword_Blades)] weapon group. The \[Short Blade\] weapon type includes the following keywords: [[Concealable](/../Keywords/Keyword_Concealable)], [[Melee](/../Keywords/Keyword_Melee)], and [[One Hand](/../Keywords/Keyword_One_Hand)].
