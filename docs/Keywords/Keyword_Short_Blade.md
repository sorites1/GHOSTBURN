# Short Blade

\[Short Blade] is a [[Weapon Type](../Keywords/Keyword_Weapon_Types.md)]. It belongs to the [[Bladed](../Keyword_Blades.md)] weapon group. The \[Short Blade\] weapon type includes the following keywords: [[Concealable](../Keywords/Keyword_Concealable.md)], [[Melee](../Keywords/Keyword_Melee.md)], and [[One Hand](../Keywords/Keyword_One_Hand.md)].
