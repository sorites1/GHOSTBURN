# Rifle

\[Rifle\] is a [[Weapon Type](/../Keywords/Keyword_Weapon_Types)]. It belongs to the [[Heavy Arms](/../Keywords/Keyword_Heavy_Arms)] weapon group. The \[Rifle\] weapon type includes the following keywords: [[Firearm](/../Keywords/Keyword_Firearm)], [[Recoil](/../Keywords/Keyword_Recoil)], and [[Two Hands](/../Keywords/Keyword_Two_Hands)].
