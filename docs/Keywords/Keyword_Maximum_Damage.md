# Maximum Damage

In a [[Damage Array](../Keywords/Keyword_Damage_Array.md)], the last number listed is the \[Maximum Damage\] value. This number is 5x the \[Minimum Damage](../[Keywords/Keyword_Minimum_Damage.md)] value. For example, in the \[Damage Array\], 5/10/25, the 25 is the \[Maximum Damage\] value.
