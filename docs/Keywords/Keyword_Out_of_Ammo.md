# Out of Ammo

Some tricks result in you running \[Out of Ammo\] on a Face or Glitch. If you are using a [[Firearm](/../Keywords/Keyword_Firearm)] when you run \[Out of Ammo\], you must spend 1 AP to use that weapon again. If you are controlling a [[Drone](/../Keywords/Keyword_Drone)] when you run \[Out of Ammo\], you must recover the drone and then spend 1 AP to use its weapon again.
