# Attitude

\[Attitude\] is an NPC trait. It represents how an NPC feels about a particular person. \[Attitude\] can be thought of as a spectrum with three values: [[Friendly](../../Lexicon/Keywords/Keyword_Friendly.md)], [[Neutral](../../Lexicon/Keywords/Keyword_Neutral.md)], and [[Hostile](../../Lexicon/Keywords/Keyword_Hostile.md)]. 

Some tricks cause a target's \[Attitude\] to improve or worsen. When an \[Attitude\] improves, it goes from \[Hostile\] to \[Neutral\] to \[Friendly\]. When it worsens, it goes from \[Friendly\] to \[Neutral\] to \[Hostile\].
