# Unarmed Strike

Indicates an [[Attack](../../Lexicon/Keywords/Keyword_Attack.md)] made without weapons, such as a punch or kick. See the [[Unarmed Strike](../../Skills_and_Tricks/Brawl/Unarmed_Strike.md)] trick for more information. 

The [[Damage Array](../../Lexicon/Keywords/Keyword_Damage_Array.md)] for \[Unarmed Strike\] uses your Strength score for [[Minimum Damage](../../Lexicon/Keywords/Keyword_Minimum_Damage.md)], Strength x 2 for [[Normal Damage](../../Lexicon/Keywords/Keyword_Normal_Damage.md)], and Strength x 5 for [[Maximum Damage](../../Lexicon/Keywords/Keyword_Maximum_Damage.md)].
