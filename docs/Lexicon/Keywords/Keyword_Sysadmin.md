# Sysadmin

You have \[Sysadmin\] access on a system to which multiple devices of the same type are connected. For example, you may be a \[Sysadmin\] on a security camera system, an electronic lock system, or a terminal system. \[Sysadmin\] access grants you [[Admin](../../Lexicon/Keywords/Keyword_Admin.md)] access to all connected devices. For example, \[Sysadmin\] access to a security camera system gives you \[Admin\] access to all its security cameras.
