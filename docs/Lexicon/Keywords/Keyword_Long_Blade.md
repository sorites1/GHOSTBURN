# Long Blade

\[Long Blade] is a [[Weapon Type](../../Lexicon/Keywords/Keyword_Weapon_Types.md)]. It belongs to the [[Bladed](../../Lexicon/Keywords/Keyword_Blades.md)] weapon group. The \[Long Blade\] weapon type includes the following keywords: [[Melee](../../Lexicon/Keywords/Keyword_Melee.md)] and [[Two Hands](../../Lexicon/Keywords/Keyword_Two_Hands.md)].
