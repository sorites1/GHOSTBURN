# Weapon Groups

A \[Weapon Group\] is a collection of [[Weapon Types](../../Lexicon/Keywords/Keyword_Weapon_Types.md)]. 

There are four \[Weapon Groups\]: [[Blades](../../Lexicon/Keywords/Keyword_Blades.md)], [[Blunt Force](../../Lexicon/Keywords/Keyword_Blunt_Force.md)], [[Small Arms](../../Lexicon/Keywords/Keyword_Small_Arms.md)], and [[Rifles](../../Lexicon/Keywords/Keyword_Rifles.md)]. 

Weapon Groups are used by [Armor](../../Lexicon/Keywords/Keyword_Armor.md).
