# Rifle

\[Rifle\] is both a [[Weapon Type](../../Lexicon/Keywords/Keyword_Weapon_Types.md)]. The \[Rifle\] weapon type includes the following keywords: [[Firearm](../../Lexicon/Keywords/Keyword_Firearm.md)], [[Recoil](../../Lexicon/Keywords/Keyword_Recoil.md)], and [[Two Hands](../../Lexicon/Keywords/Keyword_Two_Hands.md)].
