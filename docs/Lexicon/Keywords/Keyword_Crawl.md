# Crawl

You can use \[Crawl\] to [[Move](../../Lexicon/Keywords/Keyword_Move.md)] by spending 2 AP to go somewhere [[Nearby](../../Lexicon/Keywords/Keyword_Nearby.md)]. If you were not already [[Prone](../../Lexicon/Conditions/Condition_Prone.md)], you automatically become \[Prone\] when you use \[Crawl\]. You cannot use \[Crawl\] to \[Move\] to a location [[Far](../../Lexicon/Keywords/Keyword_Far.md)] from your original position.
