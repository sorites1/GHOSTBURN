# Optimal Range

A [[Firearm](../../Lexicon/Keywords/Keyword_Firearm.md)] has an attribute called \[Optimal Range\], which has one of the [[Range](../../Lexicon/Keywords/Keyword_Range.md)] values: [[Close](../../Lexicon/Keywords/Keyword_Close.md)], [[Nearby](../../Lexicon/Keywords/Keyword_Nearby.md)], and [[Far](../../Lexicon/Keywords/Keyword_Far.md)]. If you make an [[Attack](../../Lexicon/Keywords/Keyword_Attack.md)] against a target at a \[Range\] other than the \[Optimal Range\], raise the DL by one.
