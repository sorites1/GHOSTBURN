# Unarmed

\[Unarmed\] is a [[Weapon Type](../../Lexicon/Keywords/Keyword_Weapon_Types.md)]. It belongs to the [[Blunt Force](../../Lexicon/Keywords/Keyword_Blunt_Force.md)] weapon group. The \[Unarmed\] weapon type includes the following keywords: \[Brawl\] and [[Two Hands](../../Lexicon/Keywords/Keyword_Two_Hands.md)].
