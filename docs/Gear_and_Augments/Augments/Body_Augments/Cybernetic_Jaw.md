# Cybernetic Jaw \[Augment\]

*A Cybernetic Jaw is a replacement for your natural jaw.*

|   Cost   | Augment                  | Mesh Cost | Mod Slots | Keywords |
| :------: | ------------------------ | :-------: | :-------: | :------- |
| 5,000 ⌆  | Cybernetic Jaw (Base)    |     1     |     0     | -        |
| 10,000 ⌆ | Cybernetic Jaw (Level 1) |     1     |     1     | -        |
| 20,000 ⌆ | Cybernetic Jaw (Level 2) |     1     |     2     | -        |
| 40,000 ⌆ | Cybernetic Jaw (Level 3) |     1     |     3     | -        |

Cybernetic Jaw (Base) is a replacement only. It performs all the functions of a natural jaw. 

## Augment Mods

The following augments have the [[Mod](../../../Lexicon/Keywords/Keyword_Mod.md)] keyword and can be installed in a Cybernetic Jaw with a free Mod Slot. Each \[Mod\] augment takes one Mod Slot. Mods: [Poison Reservoir](../../../Gear_and_Augments/Augments/Combat_Augments/Poison_Reservoir.md) 

## Titanium Structure Mod

You can install the Titanium Structure Mod in a free Mod Slot in Cybernetic Jaw. Titanium Structure is treated as [[Armor](../../../Lexicon/Keywords/Keyword_Armor.md)] rated against [[Blunt Force](../../../Lexicon/Keywords/Keyword_Blunt_Force.md)] damage dealt by [Unarmed Strike](../../../Skills_and_Tricks/Brawl/Unarmed_Strike.md).

|   Cost   | Mod                | Keywords                                                                 |
| :------: | ------------------ | ------------------------------------------------------------------------ |
| 60,000 ⌆ | Titanium Structure | [[Armor](../../../Lexicon/Keywords/Keyword_Armor.md)], [[Mod](../../../Lexicon/Keywords/Keyword_Mod.md)] |

## Presence Mod

You can install a Presence Mod in a free Mod Slot in Cybernetic Jaw to permanently raise your Presence, as shown below. You may only install this \[Mod\] once.

|   Cost   | Mod               | Keywords                                                                               |
| :------: | ----------------- | -------------------------------------------------------------------------------------- |
| 60,000 ⌆ | +1 Presence (Aug) | [[Aug Increase](../../../Lexicon/Keywords/Keyword_Aug_Increase.md)], [[Mod](../../../Lexicon/Keywords/Keyword_Mod.md)] |

You should update your Deception, Intimidate, and Persuade skills to use your new Presence score.
