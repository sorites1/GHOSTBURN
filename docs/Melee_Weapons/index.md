# Melee Weapons

Melee weapons have a Base Value which is added to the character's Strength score when calculating its [[Damage Array](/../Gear_and_Augments/GaA_Damage_Array)], as shown below. 

| Minimum Damage        | Normal Damage               | Maximum Damage              |
| --------------------- | --------------------------- | --------------------------- |
| Strength + Base Value | (Strength + Base Value) x 2 | (Strength + Base Value) x 5 |

For example, a character with a Strength score of 3 is wielding a Switchblade, which has a Base Value of 1. \[Minimum Damage\] is 4, \[Normal Damage\] is 8, and \[Maximum Damage\] is 20. The [Melee Strike](/../Skills_and_Tricks/Trick_Melee_Core_Melee_Strike) \[Damage Array\] for this weapon when used by this character is 4/8/20.
