## Manipulate Object \[Signature Trick\]

| Difficulty Level (DL) | Skill    | Keywords                                                                                                                               |

*You use your drone's manipulator arm to open doors, use a keypad, or pick up an object, just like you would use your own.*

To use Manipulate Object, your \[[[Keyword_Drone|Drone]]\] must be outfitted with a \[[[Keyword_Manipulator_Arm|Manipulator Arm]]\]. When you use Manipulate Object, state the action you want to perform, such as *I want use my drone to type the security code into the keypad*, or *I want my drone to pick up the pistol and put it in its cargo bay*. 

| Outcome      | Description                                                           |
| ------------ | --------------------------------------------------------------------- |

| Fade         | You use the \[[[Keyword_Manipulator_Arm\|Manipulator Arm]]\] successfully and you are \[Confused\] |
| Success      | You use the \[Manipulator Arm\] successfully                          |
| Cool Success | You use the \[Manipulator Arm\] successfully and you are \[[[Condition_Inspired\|Inspired]]\] |
