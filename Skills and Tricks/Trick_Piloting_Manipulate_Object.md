## Manipulate Object \[Signature Trick\]

| Difficulty Level (DL) | Skill    | Keywords                                                                                                                               |
| --------------------- | -------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| Hard (20)             | Piloting | \[Manipulator Arm\], \[Concentrate\], \[Control Deck\], \[Datajack\], \[Drone\], \[Physical\], \[Two Hands\], \[Visual\], \[Wireless\] |

*You use your drone's manipulator arm to open doors, use a keypad, or pick up an object, just like you would use your own.*

To use Manipulate Object, your \[Drone\] must be outfitted with a \[Manipulator Arm\]. When you use Manipulate Object, state the action you want to perform, such as *I want use my drone to type the security code into the keypad*, or *I want my drone to pick up the pistol and put it in its cargo bay*. 

| Outcome      | Description                                                           |
| ------------ | --------------------------------------------------------------------- |
| Glitch       | You are unsuccessful and \[Confused\]                                 |
| Fade         | You use the \[Manipulator Arm\] successfully and you are \[Confused\] |
| Success      | You use the \[Manipulator Arm\] successfully                          |
| Cool Success | You use the \[Manipulator Arm\] successfully and you are \[Inspired\] |
