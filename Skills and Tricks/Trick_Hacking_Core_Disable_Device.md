## Disable Device \[Core Trick\]

| Difficulty Level (DL) | Skill | Keywords                                                             |
| --------------------- | ----- | -------------------------------------------------------------------- |
| \[Security\]          | Tech  | \[Fine Control\], \[Physical\], \[Tools\], \[Two Hands\], \[Visual\] |

*You inspect the device's electronic guts and make a calculated choice to snip one of the wires. A second later, the maglock disengages, and the door opens with a hiss.* 

Disable Device is a hands-on activity that requires you to directly interact with the target's hardware. If you do not have proper \[Tools\], raise the DL by one.

| Outcome      | Description                                                            |
| ------------ | ---------------------------------------------------------------------- |
| Glitch       | You fail, have \[Vision Impairment\], and trigger a \[Security Alert\] |
| Fade         | The device is \[Disabled\] and you have \[Vision Impairment\]          |
| Success      | The device is \[Disabled\]                                             |
| Cool Success | The device is \[Disabled\] and you gain a \[Secret\]                   |
