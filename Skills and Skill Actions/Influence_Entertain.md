## Entertain \[Skill Action\]

| Target                | Base Difficulty Level | Duration      | Keywords                                          |
| --------------------- | --------------------- | ------------- | ------------------------------------------------- |
| A person or \[Group\] | WITS                  | A few minutes | \[Attitude\], \[Language\], \[Social\], \[Vocal\] |

Entertain those around you by with jokes, anecdotes, or a song. Your words can improve the target's \[Attitude\] and prime them for your the next \[Social\] skill action. 

| Failure          | Description                                                             |
| ---------------- | ----------------------------------------------------------------------- |
| Critical Failure | As Standard Failure but the target's \[Attitude\] also drops one level. |
| Standard Failure | The target is not entertained and you suffer a \[Setback\].             |

| Success       | Description                                                                                               |
| ------------- | --------------------------------------------------------------------------------------------------------- |
| Minor (6)     | The target is entertained and their \[Attitude\] to go up one level. Or take a \[Second Chance\] instead. |
| Moderate (10) | The target is entertained and their \[Attitude\] becomes Friendly.                                        |
| Major (14)    | As Moderate (10) and the GM offers to make the target \[Shaken\] or \[Emotional\].                        |
| Severe (18)   | As Major (14), but the target is both \[Shaken\] and \[Emotional\].                                       |
| Extreme (22)  | As Severe (18) but target becomes your \[Ally\].                                                          |
