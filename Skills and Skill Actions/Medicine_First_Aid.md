## First Aid \[Core Skill Action\]

| Difficulty Level (DL) | Skill    | Keywords                                              |
| --------------------- | -------- | ----------------------------------------------------- |
| Stabilization DL      | Medicine | \[Physical\], \[Supplies\], \[Two Hands\], \[Visual\] |

First Aid is used to apply \[Stabilize\] on the wounded or injured. \[Stabilize\] removes several conditions, such as \[Traumatized\], \[Critical Condition\], and \[Dying\]. If you use First Aid without \[Supplies\] raise the DL by one. 

| Outcome      | Description                                            |
| ------------ | ------------------------------------------------------ |
| Glitch       | Deal 2 damage and take a \[Second Chance\]             |
| Fade         | \[Stabilized\] and \[Stress\] vs. \[Stabilization DL\] |
| Success      | \[Stabilized\] and heal 2 HP                           |
| Cool Success | \[Stabilized\] and heal 2 HP + Medicine rank           |
