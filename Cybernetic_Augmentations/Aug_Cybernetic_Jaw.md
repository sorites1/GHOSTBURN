# Cybernetic Jaw \[Augment\]

*A Cybernetic Jaw is a replacement for your natural jaw.*

|   Cost   | Augment                  | Mesh Cost | Mod Slots | Keywords |
| :------: | ------------------------ | :-------: | :-------: | :------- |
| 5,000 ⌆  | Cybernetic Jaw (Base)    |     1     |     0     | -        |
| 10,000 ⌆ | Cybernetic Jaw (Level 1) |     1     |     1     | -        |
| 20,000 ⌆ | Cybernetic Jaw (Level 2) |     1     |     2     | -        |
| 40,000 ⌆ | Cybernetic Jaw (Level 3) |     1     |     3     | -        |

Cybernetic Jaw (Base) is a replacement only. It performs all the functions of a natural jaw. 

## Augment Mods

The following augments have the \[[[Keywords/Keyword_Mod|Mod]]\] keyword and can be installed in a Cybernetic Jaw with a free Mod Slot. Each \[Mod\] augment takes one Mod Slot. Mods: [[Cybernetic_Augmentations/Aug_Poison_Reservoir|Poison Reservoir]] 

## Titanium Structure Mod

You can install the Titanium Structure Mod in a free Mod Slot in Cybernetic Jaw. Titanium Structure is treated as \[[[Keywords/Keyword_Armor|Armor]]\] rated against \[[[Keywords/Keyword_Blunt_Force|Blunt Force]]\] damage dealt by [[Skills_and_Tricks/Trick_Brawl_Core_Unarmed_Strike|Unarmed Strike]].

|   Cost   | Mod                | Keywords                                                                 |
| :------: | ------------------ | ------------------------------------------------------------------------ |
| 60,000 ⌆ | Titanium Structure | \[[[Keywords/Keyword_Armor\|Armor]]\], \[[[Keywords/Keyword_Mod\|Mod]]\] |

## Presence Mod

You can install a Presence Mod in a free Mod Slot in Cybernetic Jaw to permanently raise your Presence, as shown below. You may only install this \[Mod\] once.

|   Cost   | Mod               | Keywords                                                                               |
| :------: | ----------------- | -------------------------------------------------------------------------------------- |
| 60,000 ⌆ | +1 Presence (Aug) | \[[[Keywords/Keyword_Aug_Increase\|Aug Increase]]\], \[[[Keywords/Keyword_Mod\|Mod]]\] |

You should update your Deception, Intimidate, and Persuade skills to use your new Presence score.
