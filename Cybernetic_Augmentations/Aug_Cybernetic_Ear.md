# Cybernetic Ear \[Augment\]

*A Cybernetic Ear is a full replacement of one of your existing ears.*

|   Cost   | Augment                  | Mesh Cost | Mod Slots | Keywords |
| :------: | ------------------------ | :-------: | :-------: | -------- |
| 7,000 ⌆  | Cybernetic Ear (Base)    |     2     |     0     | -        |
| 10,000 ⌆ | Cybernetic Ear (Level 1) |     2     |     1     | -        |
| 20,000 ⌆ | Cybernetic Ear (Level 2) |     2     |     2     | -        |
| 30,000 ⌆ | Cybernetic Ear (Level 3) |     2     |     3     | -        |

Cybernetic Ear (Base) is a replacement only. It performs all the functions of a natural ear. 

If you have two Cybernetic Ears, they may be of different levels and each has its own Mod Slots. 

## Clearsound Mod

You can install the Clearsound Mod in a free Mod Slot in a Cybernetic Ear, as shown below. You may only install this \[Mod\] once. Clearsound works by automatically identifying and amplifying important sounds while filtering out unwanted noise. With the Clearsound Mod installed, you cannot gain \[[[Conditions/Condition_Hearing_Impairment|Hearing Impairment]]\]. 

|   Cost   | Mod        | Keywords                          |
| :------: | ---------- | --------------------------------- |
| 15,000 ⌆ | Clearsound | \[[[Keywords/Keyword_Mod\|Mod]]\] |

## Sound Dampener Mod

You can install the Sound Dampender Mod in a free Mod Slot in a Cybernetic Ear, as shown below. You may only install this \[Mod\] once. Sound Dampener works by automatically dampening sudden, loud sounds so you hear them at a manageable level. With the Sound Dampender Mod installed, you cannot gain [[Conditions/Condition_Deafened|Deafened]]. 

|   Cost   | Mod            | Keywords                          |
| :------: | -------------- | --------------------------------- |
| 15,000 ⌆ | Sound Dampener | \[[[Keywords/Keyword_Mod\|Mod]]\] |

## Voice Stress Analyzer Mod

You can install the Voice Stress Analyzer Mod in a free Mod Slot in a Cybernetic Ear, as shown below. You may only install this \[Mod\] once. Voice Stress Analyzer works by automatically analyzing speech you hear. With the Voice Stress Analyzer Mod installed, when you use [[Skills_and_Tricks/Trick_Insight_Core_Sense_Lies|Sense Lies]], the DL is lowered by one. 

|   Cost   | Mod                   | Keywords                          |
| :------: | --------------------- | --------------------------------- |
| 15,000 ⌆ | Voice Stress Analyzer | \[[[Keywords/Keyword_Mod\|Mod]]\] |

## Awareness Mod

You can install an Awareness Mod in a free Mod Slot in a Cybernetic Ear to permanently raise your Awareness, as shown below. You may install this \[Mod\] multiple times, and its effects are cumulative. Each time you install a Awareness Mod, it uses one Mod Slot. 

|   Cost    | Mod                | Keywords                                                                               |
| :-------: | ------------------ | -------------------------------------------------------------------------------------- |
| 100,000 ⌆ | +1 Awareness (Aug) | \[[[Keywords/Keyword_Aug_Increase\|Aug Increase]]\], \[[[Keywords/Keyword_Mod\|Mod]]\] |

You should update your Piloting, Insight, and Perception skills to use your new Awareness score.
