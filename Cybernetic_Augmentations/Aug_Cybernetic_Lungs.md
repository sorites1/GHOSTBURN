# Cybernetic Lungs \[Augment\]

*A Cybernetic Lungs replaces both of your existing, natural lungs.* 

|   Cost   | Augment                    | Mesh Cost | Mod Slots | Keywords |
| :------: | -------------------------- | :-------: | :-------: | -------- |
| 10,000 ⌆ | Cybernetic Lungs (Base)    |     3     |     0     | -        |
| 25,000 ⌆ | Cybernetic Lungs (Level 1) |     3     |     1     | -        |
| 50,000 ⌆ | Cybernetic Lungs (Level 2) |     3     |     2     | -        |
| 80,000 ⌆ | Cybernetic Lungs (Level 3) |     3     |     3     | -        |

Cybernetic Lungs (Base) are a replacement only. They performs all the functions of natural lungs. 

## Oxygen Reservoir Mod

You can install an Oxygen Reservoir Mod in a free Mod Slot in Cybernetic Lungs, as shown below. The Oxygen Reservoir has the following effect: If you would gain \[[[Conditions/Condition_Drowning|Drowning]]\] or \[[[Conditions/Condition_Suffocating|Suffocating]]\], you do not gain them. You may only install this \[Mod\] once.

|   Cost   | Mod              | Keywords                          |
| :------: | ---------------- | --------------------------------- |
| 10,000 ⌆ | Oxygen Reservoir | \[[[Keywords/Keyword_Mod\|Mod]]\] |

## Assistive Breather Mod

You can also install an Assistive Breather Mod in a free Mod Slot in Cybernetic Lungs, as shown below. The Assistive Breather Mod has the following effect: You gain a +1 \[[[Keywords/Keyword_Aug_Bonus|Aug Bonus]]\] to Athletics.  You may only install this \[Mod\] once.

|  Cost   | Mod                | Keywords                                                                         |
| :-----: | ------------------ | -------------------------------------------------------------------------------- |
| 5,000 ⌆ | Assistive Breather | \[[[Keywords/Keyword_Mod\|Mod]]\], \[[[Keywords/Keyword_Aug_Bonus\|Aug Bonus]]\] |

## Toxin Filter Mod

You can also install a Toxin Filter Mod in a free Mod Slot in Cybernetic Lungs, as shown below. The Toxin Filter Mod has the following effect: You are immune to airborne toxins.  You may only install this \[Mod\] once.

|   Cost   | Mod          | Keywords                          |
| :------: | ------------ | --------------------------------- |
| 15,000 ⌆ | Toxin Filter | \[[[Keywords/Keyword_Mod\|Mod]]\] |
