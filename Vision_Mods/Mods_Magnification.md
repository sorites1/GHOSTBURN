# Magnification

\[Magnification\] is a quality that can be applied to goggles, helmets (with visor), video cameras, and cybernetic eyes. When \[Magnification\] is active, you can zoom in so you can see \[[[Keywords/Keyword_Far|Far]]\] away more easily. When used with the Firearms skill, \[Magnification\] acts as a \[[[Weapon_Mods/Mods_Scope|Scope]]\] except it is not restricted to only Heavy Pistols and Rifles.
