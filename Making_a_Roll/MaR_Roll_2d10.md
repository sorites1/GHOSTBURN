# Roll 2d10

After you and the GM finish adjusting the DL, you roll a **2d10** (two ten-sided dice) and add the appropriate **ability score** along with any **bonuses** to get the result. 

| Making a roll...                                                                  |
| --------------------------------------------------------------------------------- |
| 2d10 + Ability Score + Skill Bonus + Gear Bonus + Aug Bonus + Luck Bonus = Result |
