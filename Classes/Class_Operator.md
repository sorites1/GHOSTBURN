## Operator

The Operator description. 

**Favored Ability:**
* Intelligence

**Favored Skills:**
* Hacking
* Tech
* Stealth
* Firearms
### Specializations
There are three Operator specializations.
#### Drone Demon
Master of drone warfare

**Favored Skill Actions:**
* Tech (Run Diagnostics)
* Tech (Repair Device)
* Tech (Install Enhancement)
* Tech (Pilot Drone)
* Tech (Build Prototype)
* Stealth (Hide)
* Firearms (Single Shot) 
#### Cipher Wraith
Stealthy field tech specializing in infiltration

**Favored Skill Actions:**
* Hacking (Street Hack)
* Tech (Run Diagnostics)
* Tech (Repair Device)
* Tech (Disable Device)
* Stealth (Move Silently)
* Stealth (Hide)
* Stealth (Slip Past)
#### Data Dragon
Dominates the Net, hoards data

**Favored Skill Actions:**
* Hacking (Street Hack)
* Hacking (Penetrate Firewall)
* Hacking (System Override)
* Hacking (Ghost Trace)
* Hacking (Manipulate Data)
* Stealth (Hide)
* Stealth (Move Silently)
