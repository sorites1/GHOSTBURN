# Aug Blade

\[Aug Blade] is a \[[[Keywords/Keyword_Weapon_Types|Weapon Type]]\]. It belongs to the \[[[Keyword_Blades|Bladed]]\] weapon group. The \[Aug Blade\] weapon type includes the following keywords: \[[[Keywords/Keyword_Brawl|Brawl]]\], \[[[Keywords/Keyword_Melee|Melee]]\], \[[[Keywords/Keyword_One_Hand|One Hand]]\], and \[[[Keywords/Keyword_Physical|Physical]]\]. 

Weapons with this \[Weapon Type\] use the Brawl skill to \[[[Keywords/Keyword_Attack|Attack]]\], but they use the Melee [[Gear_and_Augments/GaA_Damage_Array|Damage Array]] and can only use tricks from the Melee skill. 

For example, the [[Cybernetic_Augmentations/Aug_TGR_Claws|TGR Claws]] augment is classified as an \[Aug Blade\]. To make an \[Attack\], you would [[Playing_the_Game/PtG_Making_a_Roll|Make a Roll]] using Brawl, but you would not use the [[Skills_and_Tricks/Trick_Brawl_Core_Unarmed_Strike|Unarmed Strike]] trick. Instead, you would choose from the tricks found in the Melee skill, such as [[Skills_and_Tricks/Trick_Melee_Core_Melee_Strike|Melee Strike]], [[Skills_and_Tricks/Trick_Melee_Dazzling_Strike|Dazzling Strike]], or [[Skills_and_Tricks/Trick_Melee_Leaping_Strike|Leaping Strike]].
