# Weapon Types

Each weapon has a \[Weapon Type\] which grants it certain characteristics, as shown in the chart below. There are eight \[Weapon Types\] are Blade, Club, Heavy Pistol, Light Pistol, Rifle, Shotgun, SMG, and Unarmed. Each \[Weapon Type\] belongs to a \[[[Keyword_Weapon_Groups|Weapon Group]]\].

| Weapon Type                                         | Weapon Group | Weapon Type Characteristics                                                                                                                                                                                                                 |
| --------------------------------------------------- | ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Blade                                               | Bladed       |                                                                                                                                                                                                                                             |
| Club                                                | Blunt Force  |                                                                                                                                                                                                                                             |
| \[[[Keywords/Keyword_Heavy_Pistol\|Heavy Pistol]]\] | Heavy Arms   | \[[[Keywords/Keyword_Firearm\|Firearm]]\], \[[[Keywords/Keyword_Heavy_Arms\|Heavy Arms]]\], \[[[Keywords/Keyword_One_Hand\|One Hand]]\], \[[[Keywords/Keyword_Recoil\|Recoil]]\], \[[[Keywords/Keyword_Two_Handed_Grip\|Two-Handed Grip]]\] |
| \[[[Keywords/Keyword_Light_Pistol\|Light Pistol]]\] | Light Arms   | \[[[Keywords/Keyword_Concealable\|Concealable]]\], \[Firearm\], \[One Hand\], \[Two-Handed Grip\]                                                                                                                                           |
| \[[[Keywords/Keyword_Rifle\|Rifle]]\]               | Heavy Arms   | \[Firearm\], \[Recoil\], \[[[Keywords/Keyword_Two_Hands\|Two Hands]]\]                                                                                                                                                                      |
| \[[[Keywords/Keyword_Shotgun\|Shotgun]]\]           | Light Arms   | \[Firearm\], \[Recoil\], \[[[Keywords/Keyword_Spread\|Spread]]\], \[Two Hands\]                                                                                                                                                             |
| \[[[Keywords/Keyword_SMG\|SMG]]\]                   | Light Arms   | \[Firearm\], \[One Hand\], \[Recoil\], \[[[Keywords/Keyword_Spray\|Spray]]\]                                                                                                                                                                |
| Unarmed                                             | Blunt Force  |                                                                                                                                                                                                                                             |
