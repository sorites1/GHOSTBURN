# Weapon Types

Each weapon has a \[Weapon Type\] which grants it certain characteristics, as shown in the chart below. There are eight \[Weapon Types\] are Blade, Club, Heavy Pistol, Light Pistol, Rifle, Shotgun, SMG, and Unarmed. Each \[Weapon Type\] belongs to a \[[[Keywords/Keyword_Weapon_Group|Weapon Group]]\].

| Weapon Type  | Weapon Group | Weapon Type Characteristics                                                                                                                                         |
| ------------ | ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Blade        | Bladed       |                                                                                                                                                                     |
| Club         | Blunt Force  |                                                                                                                                                                     |
| Heavy Pistol | Heavy Arms   | \[[[Keywords/Keyword_Firearm\|Firearm]]\], \[Heavy Arms] \[[[Keywords/Keyword_One_Hand\|One Hand]]\] \[[[Keywords/Keyword_Recoil\|Recoil]]\], \[[[Keywords/Keyword_Two_Handed_Grip\|Two-Handed Grip]]\] |
| Light Pistol | Light Arms   | \[[[Keywords/Keyword_Concealable\|Concealable]]\], \[Firearm\], \[One Hand\], \[Two-Handed Grip\]                                                                            |
| Rifle        | Heavy Arms   | \[Firearm\], \[Recoil\], \[[[Keywords/Keyword_Two_Hands\|Two Hands]]\]                                                                                                       |
| Shotgun      | Light Arms   | \[Firearm\], \[Recoil\], \[[[Keywords/Keyword_Spread\|Spread]]\]                                                                                                             |
| SMG          | Light Arms   | \[Firearm\], \[Recoil\], \[[[Keywords/Keyword_Spray\|Spray]]\]                                                                                                               |
| Unarmed      | Blunt Force  |                                                                                                                                                                     |
